# Calendar application
Hakell calendar application as final project for Functional Programming 1 course

Example usage:

> **note** the brackets are necessary and should be part of the input

booking a meeting: 

`Event [Meeting] happens at [Office] on 2025-11-21`

getting information about the bookings:

`Tell me about [Meeting]`

`What happens at [Office]`

`What happens near 2025-11-21`

quitting:

`Quit`
