# Phone Book
Haskell phonebook application made as part of functional programming 1

you can add names and phonenumbers

Example usage:

adding phone number:

```
add <name> <phoneType> <countryCode> <phoneNo>
add Alice WorkMobile 358123456
```
looking up a phone number:

```
find <name>
find Alice
```
quitting:
```
Quit
```
