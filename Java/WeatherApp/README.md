# WeatherApp
---
A nice and simple weather APP with interactive UI that uses the OpenWeather API to getch weather data for places and cities

Created as the Final project for Programming 3 course in October 2023

All project documentation can be found in the Documentation folder.

Does not run without your own API key

To run the application, open the command prompt and execute:

```mvn package```

and then:

```java -jar target\WeatherApp-1.0.one-jar.jar```

Note: You need to have Java and Maven installed on your system.
